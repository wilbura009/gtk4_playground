# Cosmic Agenda

A simple gtk4 application to track your todo list.

![preview](docs/img/preview.png)

## Installation

```
git clone https://github.com/wilbura009/gtk4_playground.git
```

## Usage

Compile

```
cd gtk4_playground/proj/cosmicagenda/ && make
```

Execute

```
./cosmicagenda
```
